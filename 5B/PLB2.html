Program to obtain count of digits in a number

Problem Statement

Write a code in any programming language that counts the digits in a number



Code

We write the program in Python

# We write the recursive call function.

def countDigit(n):
  if n/10 == 0:
    return 1
  return 1 + countDigit(n // 10)

n = int(input())

# Calling the Function in main
print(countDigit(n))

Testcases

Sample Input

5893568

Sample Output

7

Sample Input

46

Sample Output

2

Variables

n - We shall traverse the recursive function through this n till it satisfies the base condition.

Complexity

Time Complexity: O(n)
